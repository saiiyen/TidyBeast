## 📦 Quick Start

```bash
# Clone and setup
git clone https://github.com/saiiyen/TidyBeast.git
cd TidyBeast
npm install

# Development
npm run dev              # Start dev server (port 8080)
npm run build           # Production build
npm run preview         # Preview build
```
